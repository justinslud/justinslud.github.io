Virtual Reality Surface Viewer

What

  • Virtual reality scene to view 3d mathematical surfaces from NIST's Digital Library which is also accessible as a webpage
    • Rotation, scaling, shading, axes, selection, can be changed using Oculus Rift
  • Command line tool to convert 3d object files into other common formats

Why

  • Experimental environment for seeing how to interact with 3d objects in virtual reality for research and intuition purposes

How

  • A-Frame (JavaScript Virtual Reality Framework)
  • NodeJS (for CLI)
  • HTML and CSS (for NIST website)

Where

  • NIST website
  • Virtual Reality Scene
  • Presentation slides

When

  • Summer 2019, National Insitute of Standards and Technology (NIST)
