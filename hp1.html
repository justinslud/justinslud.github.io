Justin Slud : Hutter Prize Goal Progress Report 1

This post is to update on my project to build a text compressor. I began this project in January 2020. My goal is to get on the Hutter Prize leaderboard. It is probably too ambitious, but I will try. It has been a great learning experience regardless.

I don't expect anyone to read this or care about it or even understand my notes, but I want to hold myself accountable to writing these reports. However, I do plan on writing more general posts about data compression.

January

03

  • read c++ implementation of LZ77
  • started snippets file for c++ notes
  • learned about c++ command line arguments, g++ flags, hex to binary

04

  • wrote python scipt to get samples of enwik9 dataset
  • use python's bytes function and bitarray library to practice using bits and bytes
  • started making project folders in CodeBlocks
  • downloaded lpaq8 compressor and read some of it
  • watch Compressor Head videos
  • looked at enwik9 samples and thought about using redundancy in xml tags to code them more easily (I later abandoned this idea since most of the data lies within the text and not the tags)

05

  • read and wrote down example of Huffman coding and Burrows Wheeler transform
  • discovered encode.su, the forum about compression, but didn't understand the very technical discussions and didn't get much out of the meta-discussions
  • wrote basic c++ functions equivalent to python's split and join
  • read and followed along (actually writing their code into my own file and running it for better understanding) c++ implementation of Huffman coding
  • added a decoder for Huffman-encoded strings

  • copied and modified c++ code to display progress bar in console
  • benchmarked enwik samples on lpaq8e and wrote down compression time, compression ratio, memory used, and decompression time
  • wrote down example of LZ77 coding

  • learned about c++ templates
  • read P. Skiblinski's papers about reversible text transformations for preprocessing which improve paq8's context mixing
