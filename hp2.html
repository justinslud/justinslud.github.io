Justin Slud : Hutter Prize Goal Progress Report 2

This post is to update on my project to build a text compressor. I began this project in January 2020. My goal is to get on the Hutter Prize leaderboard. It is probably too ambitious, but I will try. It has been a great learning experience regardless.

Reflections

I definitely understand the goal better in terms of what is well established (arithmetic coding, dictionary preprocessing) and what is not (context mixing, using deep learning for language modeling) I am fortunate to have others' code to learn from and use parts of in my own projects. The amount of projects and even the complexity within cmix are extreme, which makes sense given it's state-of-the-art and contains many optimizations and hidden mathematical assumptions.

Moving Forward

I still have trouble understanding how to build an entire project from scratch, so I want to continue tinkering with c++ existing projects and trying to learn enough of keras and tensorflow to use the RNN's for character prediction. It would be great to get xeus-cling, a c++ kernel for jupyter notebooks, working, so I could quickly try new language features before using then in vscode projects which is much slower.
